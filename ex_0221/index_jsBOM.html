윈도우 위치 객체

참고포트 번호가 기본값(http의 경우 80, https의 경우 443)인 경우 대부분의 브라우저는 0 또는 아무것도 표시하지 않습니다.

cookieEnabled 속성은 쿠키가 활성화되어 있는 경우 true를 반환합니다.

appName 속성은 브라우저의 애플리케이션 이름을 반환합니다.

이상하게도 "Netscape"는 IE11, Chrome, Firefox, Fafari의 애플리케이션 이름입니다.

appCodeName 속성은 브라우저의 코드 이름을 반환합니다.

"Mozilla"는 chrome, firefox, ie, safari, opera의 애플리케이션 코드명임

product 속성은 브라우저의 제품 이름을 반환합니다.

대부분 부라우저는 제품 이름으로 "Gecko"를 반환합니다.

브라우저가 온라인인 경우 true를 반환

javaEnable()메서드는 Java가 활성화 되어있으면 true를 반환
